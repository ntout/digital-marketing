# Codex prompt — US-010: In-app campaign editing (write-back to platforms)

> Paste CONTEXT.md into this session first, then paste this prompt.
> US-003 (with write OAuth scopes), US-007 (campaign table) must be complete.

---

Read the full story in `stories/US-010-inapp-campaign-editing.md`.

## Before starting
- Confirm PlatformApiService from US-003 exists — write-back calls go through it
- OAuth connections may need to be re-authorized with write scopes. Add a "Reconnect with edit permissions" flow to the Connections page for platforms missing write scope.

## Implementation instructions

1. **Database schema** — Add `CampaignChangeLog` model using the exact fields in CONTEXT.md. This table is foundational — US-011, US-012, US-013, and all AI stories write to it.

2. **Tests first** — One test per AC:
   - AC1: GET /api/campaigns/:id/edit → returns current budget and status fields
   - AC2: Preview step returns before/after diff without making any API call
   - AC3: POST /api/campaigns/:id/changes → calls platform API, returns success/error
   - AC4: Successful change → campaignChangeLog entry created with all required fields
   - AC5: Failed platform API call → no change log entry created, original values preserved
   - AC6: Viewer/Marketer role → edit endpoint returns 403

3. **Implementation** — Build the preview + confirm pattern as a two-step API:
   - `POST /api/campaigns/:id/changes/preview` — validates and returns diff, no side effects
   - `POST /api/campaigns/:id/changes/confirm` — executes platform API call, writes log

4. **PlatformApiService** — Add write methods: `updateCampaignBudget()`, `updateCampaignStatus()`. These must handle platform-specific field mapping (each platform has different budget field names).

## File structure to create
```
src/
  prisma/schema.prisma          (add CampaignChangeLog model)
  src/services/CampaignEditService.ts
  src/routes/campaign-edits.ts
  tests/campaign-edits.test.ts
```

## Do not build
- Bulk multi-campaign edits (US-012)
- Ad copy editing (US-011)
- Creating new campaigns from scratch
